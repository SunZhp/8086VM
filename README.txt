指定cpu平台，目前只支持8086
指定中断控制器，目前只支持8259A

说明:
			==========================           ============
			|                        |           |          |
			|         CPU            | --------- |   8259A  |
			|                        |           |          |
			==========================           ============
						|
		_____________________________________
		|            |           |           |
  		|            |           |           |
	==========   ==========  ==========  ===========
	|        |   |        |  |        |  |         |
	|Keyboard|   |harddisk|  |  vgio  |  |  memory |
	|        |   |        |  |        |  |         |
	==========   ==========  ==========  ===========
                                 |     								
                                 |     
                             ==========
	                         |        |
                             |  vgui  |
                             |        |
                             ==========

	CPU: 	  指令的执行以及bios部分
	8259A:    中断控制器
	Keyboard: 键盘输入
	harddisk: 硬盘读写
	vgio:     虚拟图形输入输出接口
	memory:   内存读写
